How to Use the Tools

Drawing Tools

  • Pencil: Basic drawing tool for precise lines
  • Brush: Smooth, flowing strokes with pressure sensitivity
  • Straight Line: Draw perfect straight lines from start to end
  • Spray: Creates spray paint effect with adjustable density
  • Eraser: Removes drawn content from the canvas
  • Dotted: Creates dotted line patterns
  • Blur: Softens and blurs existing content
  • Smudge: Smears and blends colors together

Text & Content

  • Text Tool: Add customizable text with different fonts and sizes
  • Upload Image: Import images from your device
  • AI Generate: Create images using AI prompts
  • Emoji Picker: Add emojis and symbols
  • MonAnimals: Build characters using pre-made parts
Note: Pixelated mode uploads are disabled for on-chain minting due to high gas costs.

Editing Tools

  • Crop: Select and crop areas of your artwork
  • Resize: Scale and resize elements on the canvas
  • Undo/Redo: Navigate through your editing history
  • Clear: Erase the entire canvas

Color & Size

  • Color Picker: Choose any color for your tools
  • Size Slider: Adjust brush and tool sizes
  • Pixel Mode: Switch to pixelated board for pixel art
  • On-Chain Mode: Create NFTs directly on blockchain

Export & Save

  • Save PNG: Download as high-quality PNG image
  • Save JPG: Download as compressed JPG image
  • Mint NFT: Create and mint your artwork as an NFT
  • Mint On-Chain: Store artwork directly on blockchain

Pro Tips

  • Use Ctrl+Z for quick undo
  • Use the Straight Line tool for perfect lines
  • Use pixel mode for retro-style artwork
  • Connect wallet to mint your creations as NFTs
